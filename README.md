# RecipeHub 

A full-featured React recipe management application with CRUD functionality and persistent storage.

##  Features
- Create, update and delete recipes
- Add and remove favourites
- Persistent storage using localStorage
- Responsive UI using Tailwind CSS

##  Technologies Used 
- React.js
- Context API
- React Router DOM
- React Hook Form
- Tailwind CSS
- LocalStorage
